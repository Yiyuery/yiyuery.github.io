Python 库的基本使用方法

作者：玉川   2020 年 2 月 24 日  

这一篇简要介绍一下几种引入库的方式

引入外部库的前提是你已经安装了对应的库，有些库是 Python 自带的，比如math库，但是很多库并没有默认安装，所以在使用之前就需要手动安装，比如numpy

安装方法：

pip install <库名>

$ pip install numpy

# import 方法

import <库名>

如引入 math 库

import math

print(math.pi)

因为是全局引入，所以使用的时候需要在前面加上库的名称，即math.pi

有时候库的名字比较长，所以有时候为了使用方便而使用别名：

import <库名> as <别名>

import math as mt

print(mt.pi)

from <库名> import <模块名>

# from ... import ...

比如引入 math 库的 pi

from math import pi

print(pi)

这时候可以直接使用库里面的方法或者变量。如果需要引入很多的话，可以使用逗号隔开引入。也可以全部引入：

from <库名> import *

from math import *

print(pi)
