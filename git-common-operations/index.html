Git 常用操作

作者：玉川   2020 年 1 月 4 日  

这篇文章主要是整理实际项目中会使用到的操作。

# 将当前已有项目同步到托管平台

以GitHub为例，参考：GitHub帮助

我就不赘述了，这里面介绍的很详细！补充一点，如果 你的项目之前是再Coding或者其他远程托管平台的话，就需要变通一下：

# 这里把 origin 修改成了 github
git remote add github remote repository URL
# 同理，push 的时候也要使用 github
git push github master

如果提示权限不足的话，先使用cat ~/.ssh/id_rsa.pub，查看自己的SSH key，然后去 GitHub -> Settings 里面把自己的SSH key 添加进去。

# 返回历史版本

查看历史记录，然后回到自己想要的地方

# 查看日志
git log

找到自己想要回退的版本的版本号：如：commit 4ba05c7c7bf43eec1c2e24d312fee454456134bf，复制下来，输入q推出，然后：

# 回退
git reset 4ba05c7c7bf43eec1c2e24d312fee454456134bf
